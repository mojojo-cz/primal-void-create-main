## 📊 项目技术架构

### 🏗️ 系统组成
- **前端框架**：React + TypeScript
- **UI组件库**：自定义组件 + Tailwind CSS
- **后端服务**：Supabase（PostgreSQL + Edge Functions）
- **视频存储**：MinIO对象存储
- **状态管理**：React Context + Hooks

### 🔧 核心功能模块
根据代码库分析，系统包含以下8大核心模块：
1. **用户管理系统** - 7种用户类型，完整权限控制
2. **课程管理系统** - 线上网课和线下科目管理
3. **视频管理系统** - 智能缓存和播放优化
4. **排课管理系统** - 智能排课和冲突检测
5. **激活密钥系统** - 账号升级和权限管理
6. **班级管理系统** - 多对多关系和权限分离
7. **学习进度系统** - 精确追踪和智能续播
8. **场地管理系统** - 教室资源和设备管理
